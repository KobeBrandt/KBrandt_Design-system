Typography

The typography settings define the visual style of text elements across the application, ensuring readability and consistency.

Table of Contents

  1. Font Family
  2. Line Height
  3. Font Size
  4. Headings

Font Family

The font family defines the typeface used throughout the application, contributing to the overall visual identity.

Example

This text uses the "DM Sans" font family.

Usage

  • font-family: Sets the primary font to "DM Sans" with a fallback to sans-serif.

Line Height

The line height determines the spacing between lines of text, enhancing readability.

Example

This text has a line height of 1.6, improving readability.

Usage

  • line-height: Sets the line height to 1.6 for optimal spacing between lines.

Font Size

The font size defines the base size of text elements, ensuring consistency across the application.

Example

This text has a base font size of 1em.

Usage

  • font-size: Sets the base font size to 1em, allowing for scalable text sizing.

Headings

Headings are styled to provide clear hierarchy and structure to the content.

Example

This is an H2 Heading

Usage

  • h2 Margin Top: Sets the top margin to 2rem for spacing above the heading.
  • h2 Margin Bottom: Sets the bottom margin to 0.5rem for spacing below the heading.

Code Example

Typography Settings

:root {
    font-family: "DM Sans", sans-serif;
    line-height: 1.6;
    font-size: 1em;
    h2 {
        margin-top: 2rem;
        margin-bottom: 0.5rem;
    }
}
