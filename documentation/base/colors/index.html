Color Variables

The color variables define the visual language of the application, ensuring consistency and coherence across different components and themes.

Table of Contents

  1. Primary Colors
  2. Light Mode
  3. Dark Mode
  4. Combination
  5. Additional Colors

Primary Colors

Primary colors are used for the main branding elements and interactive components.

Examples

Usage

  • --primary: Main brand color used for primary actions and highlights.
  • --primary-text: Text color used on primary backgrounds.
  • --error: Color used to indicate errors or issues.
  • --alert: Color used for alerts and notifications.
  • --disabled: Color used for disabled elements.

Light Mode

Light mode colors are used to create a bright and clean interface.

Examples

Usage

  • --light-mode-text: Text color used in light mode.
  • --light-mode-bg: Background color used in light mode.

Dark Mode

Dark mode colors are used to create a low-light interface, reducing eye strain.

Examples

Usage

  • --dark-mode-text: Text color used in dark mode.
  • --dark-mode-bg: Background color used in dark mode.

Combination

Combination variables are used to switch between light and dark modes seamlessly.

Examples

Usage

  • color-scheme: Defines the color scheme supporting both light and dark modes.
  • --text: Text color that adapts to the current mode.
  • --bg: Background color that adapts to the current mode.

Additional Colors

Additional colors are used for various UI elements to maintain visual hierarchy and contrast.

Examples

Usage

  • --white: Pure white color used for backgrounds and text.
  • --light-gray: Light gray color used for subtle backgrounds and borders.

Code Example

Primary Colors

:root {
    --primary: #5d5bda;
    --primary-text: #ffffff;
    --error: #b50000;
    --alert: #c1da0b;
    --disabled: #494949;
}

Light Mode

:root {
    --light-mode-text: #000000;
    --light-mode-bg: #ffffff;
}

Dark Mode

:root {
    --dark-mode-text: #ffffff;
    --dark-mode-bg: #404040;
}

Combination

:root {
    color-scheme: light dark;
    --text: light-dark(var(--light-mode-text), var(--dark-mode-text));
    --bg: light-dark(var(--light-mode-bg), var(--dark-mode-bg));
    color: var(--text);
    background-color: var(--bg);
}

Additional Colors

:root {
    --white: #ffffff;
    --light-gray: #d6d6d6;
}
