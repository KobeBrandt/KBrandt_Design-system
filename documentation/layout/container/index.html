Container Layout

This documentation provides an overview of the container layout styles used in the design system. These styles ensure consistent spacing and alignment across the application.

Table of Contents

  1. Container
  2. Container Detail

Container

The .container class is a general-purpose container that centers content and provides responsive padding.

Properties

  • max-width: Limits the container's width to 80rem for better readability on larger screens.
  • margin-inline: Centers the container horizontally within its parent.
  • padding-inline: Adds horizontal padding of 1rem for spacing within the container.

Container Detail

The .container-detail class is a narrower container designed for detailed content, such as forms or specific sections.

Properties

  • max-width: Limits the container's width to 40rem for a more focused layout.

Example

This is an example of a container-detail.

Usage

  • Use the .container class for general layout purposes, such as wrapping main content.
  • Use the .container-detail class for smaller, detailed sections requiring a narrower layout.
