Container

The Container used in the application. These components are designed to manage layout and spacing, ensuring a consistent look and feel across different pages.

Table of Contents

  1. Container
  2. Container Detail

Container Component

The Container Component is used to center content and provide horizontal padding, ensuring that the content is well-spaced and aligned within the viewport.

CSS

.container {
    max-width: 80rem;
    margin-inline: auto;
    padding-inline: 1rem;
}

Container Detail Component

The Container Detail Component is used for narrower content areas, such as detailed views or forms, to focus user attention on specific content.

CSS

.container-detail {
    max-width: 40rem;
}
