Aside

The Aside layout used in the application. These components are designed to manage layout and spacing, ensuring a consistent look and feel across different pages.

Table of Contents

  1. Aside Right Component
  2. Aside Left Component

Aside Right Component

The Aside Right Component is used to create a layout where the main content spans two columns, and the aside content is positioned in the third column.

CSS

.aside-right {
    grid-template-columns: repeat(3, 1fr);
    grid-column-gap: 3rem;
}
.aside-right main {
    grid-column: 1 / 3;
}
.aside-right aside {
    grid-column: 3;
}
@media (min-width: 46.875em) {
    .aside-right {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
    }
}

Aside Left Component

The Aside Left Component is used to create a layout where the aside content is positioned in the first column, and the main content spans the second and third columns.

CSS

.aside-left {
    grid-column-gap: 1rem;
}
.aside-left main {
    grid-column: 2 / 4;
}
.aside-left aside {
    order: -1;
    grid-column: 1;
}
@media (min-width: 46.875em) {
    .aside-left {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
    }
}
