Reset

The reset styles used in the design system. These styles ensure a consistent starting point for styling across different browsers by removing default margins and padding.

Table of Contents

  1. Reset Styles

Reset Styles

The reset styles for the `html` and `body` elements remove default margins and padding, providing a clean slate for consistent styling across the application.

CSS

html,
body {
    margin: 0;
    padding: 0;
}
