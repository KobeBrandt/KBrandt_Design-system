Global Box-Sizing

The global box-sizing rule ensures that all elements, including pseudo-elements, use the border-box model. This makes it easier to manage element dimensions, including padding and borders.

CSS

*, *::before, *::after {
    box-sizing: border-box;
}

Main Element

The main element is styled to have a minimum height of 80% of the viewport height. This ensures that the main content area occupies a significant portion of the screen, enhancing the layout's visual structure.

CSS

main {
    min-height: 80vh;
}
