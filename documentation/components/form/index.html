Form

The Form Components used in the application. Each component is designed to enhance user experience and provide a consistent look and feel.

Form Component

The Form Component is designed to provide a structured and responsive layout for forms. It includes styles for various form elements.

CSS

.form {
    display: flex;
    flex-direction: column;
}

Form Item

The Form Item is a sub-component used within the form to structure individual form elements.

CSS

.form .form-item {
    display: flex;
    flex-direction: column;
    gap: 0.1rem;
    margin-bottom: 1.75em;
}

Form Combined

The Form Combined component is used to display multiple form elements side by side.

CSS

.form .form-combined {
    display: flex;
    gap: 1rem;
    width: 100%;
}
.form .form-combined div {
    width: 25%;
}
.form .form-combined .large {
    width: 75%;
}

Inputs

Styles for input elements within the form, including text inputs and textareas.

CSS

.form input,
.form textarea {
    font-size: 1rem;
    padding: 0.5em;
    width: 90%;
    width: calc(100% - 1.5em);
    border-radius: 0.2rem;
}

Fieldset

The Fieldset component is used to group related form elements together.

CSS

.form fieldset {
    border: 0;
    padding: 0;
}
.form .radiobutton,
.form .checkbox {
    display: flex;
    flex-direction: row;
    align-items: center;
    gap: 0.5em;
    margin-bottom: 0.5em;
}
.form .radiobutton input,
.form .checkbox input {
    width: 3%;
}
.form .radiobutton label,
.form .checkbox label {
    width: auto;
    font-weight: normal;
}

Button

Styles for buttons within the form.

CSS

.form .button {
    width: 25%;
    font-size: 1em;
}

Error

Styles for displaying error messages within the form.

CSS

.form .error {
    display: block;
    font-weight: normal;
    margin-top: 0.25em;
    color: var(--error);
    margin: 0;
    padding: 0;
}

Disabled

Styles for disabled form elements.

CSS

.form .disabled {
    color: var(--disabled);
    border-color: var(--disabled);
}
.form .disabled:hover input {
    cursor: not-allowed;
}
