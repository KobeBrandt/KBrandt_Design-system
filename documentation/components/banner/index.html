The Banner Component used in the application. The banner is designed to display images in a responsive and visually appealing manner.

  1. Banner Component

Banner Component

The Banner Component is used to display images that cover the full width of their container while maintaining a fixed height. This ensures a consistent look across different devices.

CSS

.banner {
    width: 100%;
    height: 25rem;
    object-fit: cover;
}
