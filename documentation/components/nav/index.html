Nav Components

This documentation provides an overview of the navigation components used in the application. Each component is designed to enhance user experience and provide easy access to different sections of the application.

Nav with logo

The "Nav with Logo" component is used to display a navigation bar that includes a logo. This component is ideal for branding and providing quick access to different sections of the application.

Usage

  • Logo: Include a logo image within an anchor tag to make it clickable.
  • Navigation Items: Use an unordered list to display navigation items. Each item should be wrapped in an anchor tag for linking.
  • Styling: Apply the .nav class to style the navigation bar. Use Flexbox for alignment.

Nav with name

The "Nav with Name" component is used to display a navigation bar that includes a name or title. This component is useful for personal branding or displaying the name of a section.
