Navigation Components Documentation

The Navigation Components used in the application. Each component is designed to enhance user experience and provide a consistent look and feel.

Table of Contents

Nav Component

The Nav Component is designed to provide a flexible and responsive navigation bar. It includes styles for links, images, and responsive behavior.

CSS

.nav {
    display: flex;
    background-color: var(--primary);
}
.nav ul {
    color: var(--primary-text);
    list-style: none;
    width: 100%;
    font-weight: bold;
}
.nav ul li {
    font-size: 110%;
    border-bottom: 0.1rem solid var(--primary-text);
}
.nav ul li a {
    display: block;
    text-decoration: none;
    color: inherit;
}
.nav ul li a:hover {
    text-shadow: 0rem 0rem 0.2rem var(--light-mode-text);
}
.nav ul .current {
    text-decoration: underline;
}
@media (min-width: 46.875em) {
    .nav img {
        width: 5rem;
    }
    .nav ul {
        justify-content: right;
        display: flex;
        gap: 1rem;
    }
    .nav ul li {
        align-content: center;
        border-bottom: none;
    }
}

Nav Name Component

The Nav Name Component is used to display the name in the navigation bar with specific styles for different screen sizes.

CSS

.nav-name {
    flex-direction: column;
}
.nav-name p {
    font-weight: bold;
    text-align: center;
    color: var(--white);
    font-size: 2rem;
    margin-bottom: 0;
}
.nav-name ul {
    padding: 0;
}
.nav-name ul li {
    text-align: center;
}
.nav-name p:hover {
    text-shadow: 0rem 0rem 0.2rem var(--light-mode-text);
    text-decoration: underline;
}
@media (min-width: 46.875em) {
    .nav-name {
        display: flex;
        flex-direction: row;
    }
    .nav-name p {
        margin-top: 1rem;
        margin-bottom: 1rem;
        text-align: left;
        transition: text-shadow 0.3s ease-in-out;
        transition: text-decoration 0.3s ease-in-out;
    }
}

Nav Breadcrumb Component

The Nav Breadcrumb Component provides a styled breadcrumb navigation to help users understand their current location within the application.

CSS

.nav-breadcrumb {
    list-style: none;
    padding: 0;
    margin-top: 0;
    display: flex;
    background-color: var(--primary);
    margin-bottom: 0;
    padding-bottom: 0.5rem;
}
.nav-breadcrumb li {
    margin-top: 0;
    color: var(--white);
    display: flex;
    align-items: center;
}
.nav-breadcrumb li a {
    color: inherit;
}
.nav-breadcrumb li::after {
    content: ">";
    margin-inline: 0.5rem;
    text-decoration: none;
}
