Code Block

The Code Block Component used in the application. This component is designed to display code snippets in a readable and visually distinct manner.

Table of Contents

  1. Code Block Component

Code Block Component

The Code Block Component is used to display code snippets with a monospaced font, padding, and a light gray background. It ensures that code is easily distinguishable from other text.

CSS

.codeblock {
    background-color: var(--light-gray);
    padding: 1rem;
    color: var(--light-mode-text);
    font-family: monospace;
    border-radius: 0.2rem;
    overflow-x: auto;
}
