Card Components Documentation

This documentation provides an overview of the Card Components used in the application. Each component is designed to enhance user experience and provide a consistent look and feel.

Table of Contents

  1. Card Component
  2. Overlay Link

Card Component

The Card Component is designed to display content in a structured and visually appealing manner. It includes styles for various elements within the card.

CSS

.card {
    padding: 0;
    list-style: none;
    display: flex;
    flex-direction: column;
    gap: 3rem;
    width: 100%;
}
.card li {
    position: relative;
    display: flex;
    flex-direction: column;
    border-radius: 0.2rem;
    border: 0.1rem solid var(--primary);
}
.card li h3,
.card li p {
    padding: 1.5rem;
}
.card li p {
    padding-top: 0;
    margin: 0;
}
.card li h3 {
    padding-bottom: 0;
    margin: 0;
}
.card li img {
    order: -1;
    display: block;
    object-fit: cover;
    width: 100%;
    height: 50%;
    border-radius: 0.1rem 0.1rem 0 0;
}
.card li a {
    margin-left: 1.5rem;
    margin-right: 1.5rem;
    margin-top: auto;
}
@media (min-width: 46.875em) {
    .card {
        flex-direction: row;
        flex-wrap: wrap;
    }
    .card li {
        width: 30.7%;
    }
}

Overlay Link

The Overlay Link component is used to create a clickable overlay on top of card elements.

CSS

.overlay-link {
    position: absolute;
    inset: 0;
    text-indent: 100%;
    white-space: nowrap;
    overflow: hidden;
}
