Shopping Cart Component

The Shopping Cart Component is designed to display the items in a user's shopping cart with a clean and organized layout.

CSS

.shoppingCart {
    background-color: #f8f9fa;
    padding: 2rem;
    border-radius: 0.25rem;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
    height: fit-content;
}
.shoppingCart section {
    margin-top: 1rem;
}
.shoppingCart h2 {
    margin-top: 0;
}
.shoppingCart article {
    margin-bottom: 1rem;
}
.shoppingCart .big {
    font-size: 1.25rem;
    font-weight: bold;
}
.shoppingCart .name {
    display: flex;
}
.shoppingCart .name a {
    margin-inline-start: auto;
    color: inherit;
    padding-top: 1.1rem;
}
