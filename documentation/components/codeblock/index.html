Codeblock

The `codeblock` component is used to display preformatted code snippets with proper styling.

Example

<div class="codeblock">
    <p>This is a code snippet.</p>
</div>

Usage

  • Wrap your code inside a <pre> tag with the codeblock class.
  • Use the <code> tag inside the <pre> for semantic HTML.
