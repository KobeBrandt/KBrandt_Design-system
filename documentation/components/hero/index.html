Hero

The Hero Components used in the application. Each component is designed to enhance user experience and provide a consistent look and feel.

Table of Contents

  1. Hero Component
  2. Hero Image Component

Hero Component

The Hero Component is designed to create a prominent introductory section with centered text and customizable font sizes.

CSS

.hero {
    padding: 10rem;
    text-align: center;
    color: var(--white);
}
.hero .big {
    font-weight: bold;
    font-size: 4rem;
}
.hero .extra {
    font-size: 2rem;
}

Hero Image Component

The Hero Image Component is used to display a background image that spans the full width of the hero section, providing a visually engaging backdrop.

CSS

.hero-img {
    padding: 0;
    background-image: url(https://placehold.co/300x100);
    background-size: 100%;
    height: 30rem;
    align-content: center;
}
