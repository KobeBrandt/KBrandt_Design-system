Hover Effects Components

The hover effects components provide smooth transitions and visual feedback on hover, enhancing user interaction and engagement.

Table of Contents

  1. Hover Effect
  2. Hover Text Effect

Hover Effect

The hover effect component applies a box shadow on hover, providing a subtle visual cue to users. This effect is activated on larger screens to enhance the interactive experience.

Example

Hover over this element to see the effect.

Usage

  • Transition: Smooth transition for box shadow on hover.
  • Box Shadow: Applies a subtle box shadow using the --light-mode-text CSS variable.
  • Media Query: Effect is applied on screens with a minimum width of 46.875em.
  • Application: Add the .hover class to elements to enable the hover effect.

Hover Text Effect

The hover text effect component applies a text shadow on hover, enhancing the visibility and emphasis of text elements. This effect is activated on larger screens to improve readability and interaction.

Example

Hover over this text to see the effect.

Usage

  • Transition: Smooth transition for text shadow on hover.
  • Text Shadow: Applies a subtle text shadow using the --light-mode-text CSS variable.
  • Media Query: Effect is applied on screens with a minimum width of 46.875em.
  • Application: Add the .hover-text class to text elements to enable the hover effect.

Code Example

Hover Effect

<div class="hover" style="padding: 20px; background-color: var(--primary);">
    Hover over this element to see the effect.
</div>

Hover Text Effect

<div class="hover-text" style="padding: 20px; background-color: var(--primary); color: var(--white);">
    Hover over this text to see the effect.
</div>
