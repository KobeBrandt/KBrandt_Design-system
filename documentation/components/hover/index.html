Hover Component

The hover component provides hover effects for elements, enhancing user interaction and visual feedback.

Usage

  • Apply the hover class to elements to add a box-shadow effect on hover.
  • Apply the hover-text class to elements to add a text-shadow effect on hover.

Example

<div class="hover">Hover over me</div>
<p class="hover-text">Hover over this text</p>

Notes

  • Hover effects are only applied on devices with a minimum width of 46.875em (750px).
  • The transition duration for hover effects is 0.3s for a smooth animation.
