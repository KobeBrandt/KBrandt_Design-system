Footer Component

The Footer Component is used to display important information at the bottom of a webpage, such as copyright notices or links. It ensures a consistent look and feel across the application.

CSS

.footer {
    margin: 0;
    color: var(--primary-text);
    padding: 1rem;
    text-align: center;
}
.footer p {
    font-weight: bold;
    margin: 0;
}
