Stepper Component

The Stepper Component is designed to visually represent progress through a series of steps. It includes styles for step indicators and connecting lines.

CSS

.stepper {
    list-style: none;
    margin: 0;
    padding: 0;
    display: flex;
    position: relative;
    margin-bottom: 1rem;
}
.stepper li {
    position: relative;
    flex: 1;
}
.stepper li::before {
    content: "";
    aspect-ratio: 1/1;
    width: 1.5rem;
    border-radius: 50%;
    border: 0.2rem solid;
    background-color: var(--bg);
    display: block;
}
.stepper li::after {
    content: "";
    display: block;
    position: absolute;
    width: 100%;
    height: 0.1rem;
    border: 0.1rem solid;
    top: 0.69rem;
    left: 51%;
    transform: translateX(-50%);
    z-index: -1;
}
.stepper li:last-child::after {
    display: none;
}
.stepper li a {
    color: inherit;
}

Current Step

The Current Step is highlighted to indicate the user's current position in the stepper process.

CSS

.stepper .current::before {
    background-color: var(--primary);
