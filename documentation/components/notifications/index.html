Notification

The Notification Components used in the application. Each component is designed to enhance user experience and provide a consistent look and feel.

Table of Contents

  1. Notification Component
  2. Error Notification

Notification Component

The Notification Component is designed to display important messages to the user in a prominent and consistent manner.

CSS

.notification {
    border: 0.2rem solid var(--primary);
    border-radius: 0.2rem;
    padding: 0.5rem;
    font-weight: bold;
    text-align: center;
    font-size: 1.5rem;
}

Error Notification

The Error Notification is used to display error messages with a distinct border color to indicate issues.

CSS

.error {
    border-color: var(--error);
}

Alert Notification

The Alert Notification is used to display alert messages with a distinct border color to grab the user's attention.

CSS

.alert {
    border-color: var(--alert);
}
