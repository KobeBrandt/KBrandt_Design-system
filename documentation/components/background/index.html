Background Primary Component

The background-primary component is used to apply a primary background color and text color to elements, ensuring a consistent look and feel across the application.

Table of Contents

  1. Base Usage
  2. Customization

Base Usage

The base usage of the background-primary component involves applying the primary background color and white text color to an element. This is typically used for headers, navigation bars, and other prominent UI elements.

Example

This element has a primary background color and white text.

Usage

  • Background Color: Uses the --primary CSS variable to set the background color.
  • Text Color: Uses the --primary-text CSS variable to set the text color.
  • Application: Apply the .background-primary class to any element to set its background and text colors.

Customization

The background-primary component can be customized by modifying the --primary and --white CSS variables. This allows for flexibility in design while maintaining consistency across the application.

Example

Customized primary background color and white text.

Usage

  • CSS Variables: Change the values of --primary and --primary-text in your CSS to customize the background and text colors.
  • Additional Styling: Combine with other utility classes for padding, margins, and text styles to achieve the desired design.

Code Example

Base Usage

<div class="background-primary" style="padding: 20px;">
    This element has a primary background color and white text.
</div>

Customized Usage

<div class="background-primary" style="padding: 20px;">
    Customized primary background color and white text.
</div>
