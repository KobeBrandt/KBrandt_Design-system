Button Component

The button component provides a simple, interactive way to trigger actions, enhancing user engagement and streamlining navigation.

Table of Contents

  1. Base Button
  2. Button Sizes
  3. Button States
  4. Button Variants

Base Button

The base button is styled with a primary background color and centered text. It is designed to be easily customizable for different use cases.

Example

Click Me

Usage

  • Background Color: Uses the --primary CSS variable.
  • Text Decoration: None.
  • Text Alignment: Centered.
  • Color: White text using --white.
  • Padding: 0.5rem for comfortable click area.
  • Font Weight: Bold text.
  • Border Radius: Slightly rounded corners.
  • Margin: 1rem for spacing, except for left margin which is 0.
  • Border: None.
  • Media Query (min-width: 46.875em): Smooth transition for box shadow on hover.

Button Sizes

The button component includes different size variations to accommodate various design requirements.

Examples

Small Button Large Button Extra Large Button

Usage

  • Small Button (.button-small): Font size is 75% of the base button.
  • Large Button (.button-large): Font size is 125% of the base button.
  • Extra Large Button (.button-extra-large): Font size is 150% of the base button.

Button States

The button component includes a disabled state to indicate when a button is not interactive.

Example

Disabled Button

Usage

  • Disabled Button (.button-disabled):
    • Background Color: Uses the --disabled CSS variable.
    • Opacity: 90% opacity to indicate the disabled state.
    • Hover Effect: No box shadow and cursor changes to not-allowed.

Button Variants

The button component includes a secondary variant for alternative styling.

Example

Secondary Button

Usage

  • Secondary Button (.button-secondary):
    • Border: Thin solid border using --primary.
    • Background Color: Inherits from the parent element.
    • Color: Uses the --primary CSS variable for text color.
