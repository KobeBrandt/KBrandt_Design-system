Utilities

Utilities in a design system provide a set of reusable, low-level CSS classes that apply specific styles, enabling developers to quickly and efficiently customize components without writing new CSS. They promote consistency and scalability by standardizing common styles like margins, padding, colors, and typography, making it easier to maintain and update the design across the application.

  • Screen Reader Only

    The screen reader only (sr-only) class is used to hide elements visually while keeping them accessible to screen readers, enhancing accessibility for users with visual impairments.
